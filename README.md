# GhostVault

Güvenli Galeri ve Sosyal Medya İndirici (All-In-One).

## Kurulum

Bu proje [Flutter](https://flutter.dev) kullanılarak geliştirilmiştir. Sisteminizde Flutter yüklü görünmüyor. Projeyi çalıştırmak için lütfen Flutter SDK'sını kurun ve PATH'e ekleyin.

Kurulumdan sonra şu komutları çalıştırın:

```bash
flutter pub get
flutter run
```

## Özellikler

- **Güvenli Kasa**: AES-256 şifreleme ile fotoğraf ve videoları saklar.
- **Gizli Tarayıcı**: Geçmiş tutmayan özel tarayıcı.
- **Sosyal Medya İndirici**: TikTok (Filigransız), Instagram ve Spotify.
- **Acil Durum (Panic) Butonu**: Telefonu salladığınızda uygulama kapanır.
